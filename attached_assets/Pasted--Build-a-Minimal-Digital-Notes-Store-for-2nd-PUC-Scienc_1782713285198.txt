# Build a Minimal Digital Notes Store for 2nd PUC Science (Karnataka)

## Objective

Build a clean, minimal, mobile-first website for selling **2nd PUC Science (Karnataka Board)** digital notes and study resources.

The website should allow students to browse resources, purchase them securely, and instantly download PDFs after successful payment.

The project should be scalable so I can later add Engineering notes and other educational resources without redesigning the website.

---

# Technology Stack

Use:

* HTML5
* CSS3
* JavaScript (Vanilla)
* Supabase (Database, Storage, Authentication)
* Razorpay (Payment Gateway)
* Netlify or Vercel (Hosting)

Do NOT use React or any frontend framework.

Keep the project lightweight and beginner-friendly.

---
